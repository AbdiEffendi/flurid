Flurid The Fluid CSS Grid

Introducing jQuery.flurid

The jQuery.flurid plugin makes developing with Flurid easier by automatically adding classes for you. You no longer have to denote each row as "odd" or "even" and you don't have to worry about putting the "last" class on any columns, this is all done for you. Also, if you so choose, the plugin will also enable you to have equal height columns (meaning, all of the columns in a row will have the height of the tallest column in that row).

Browser Support

jQuery.flurid has been tested and verified to work on:

  • Internet Explorer (Windows) - versions 6.0 - 8.0
  • Mozilla Firefox (Windows/Linux) - versions 1.0 - 3.6

Getting jQuery.flurid

The most up-to-date version of jquery.flurid can be found on github: http://github.com/kflorence/flurid. For examples on how to use the plugin, please view the source of this file.

Known Bugs and Caveats

  • Requires the jQuery javaScript Library
  • Requires that users have a javaScript enabled web-browser and that they have javaScript enabled.
